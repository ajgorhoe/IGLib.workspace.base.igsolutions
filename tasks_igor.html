5         Instructions

 

Each task should have a title with state tags included at the end of the title. Heading style level of task title should be at least “Heading 3”. State tags should be added to task titles in the following form:

 

Priority-Status-Owner,Other_users

e.g.:

 

5.1.1.1       Search for contour plotting libraries Hig-Con-Tadej,Igor

 

It is allowed for task titles that they do not include state tags. Marks in state tags should in general be inserted in title text by referencing appropriate bookmarks (listed in tables in Section ‎5.2). User marks can be manually written instead of referencing bookmarks from Table 3, but should be consistent with names listed in the table. If some user is not listed in the table, please add a new mark in Table 3 or ask moderator to do so.

 

Individual marks are inserted by inserting bookmark references (Alt-I, N, R, choose bookmarks). An easier way might be copying another state tags, toggle field codes display (select text of the whole tag, right-click, choose “Toggle field codes” in the menu).

 

Each task can a task summary table that summarizes basic data. Table should normally be located immediately below the task title and should look something like this:

 

Assigned by (user, date):

Igor, avgust 22, 2013

 

Checked by (user, date):

Tadej, avgust 22, 2013

 

References:

 

 

Req. completion date(s):

Oct 10, 2012

 

 

The table can be easily inserted by copying it from this place or form another task that has the table difined.

 

Under the table, different task sections must be introduced by a line of normal text printed in bold and followed by corresponding text in normal style. If possible, names of task sections should be standard names from Table 4 (which are also bookmarked). Section names do not need to be inserted by referencing the corresponding bookmarks from Table 4, but names should be consistent with names listed in the table, if possible. It is allowed to use task section names that are not listed.

 

Text of individual task sections should use the Normal style. Since different users can add paragraphs to this text, it is desirable to introduce each portion of text by the appropriate user mark and date, e.g.

 

Course

Igor Apr. 21, 1012:

Updated the base module for Sheppard interpolation on SVN.
